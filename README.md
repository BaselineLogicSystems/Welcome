# Welcome Home!

## Description
Baseline Logic Systems Homepage

## Getting Started
### Prerequisites
This project requires javascript, html, css, and a web server for proper use.

### Installation
0. Install Dependencies
1. Clone, Build and Test
2. Deploy to test environment and Verify
3. Schedule and Deploy to Production

## Technical Stack

- Frontend: Vanilla HTML5, Modern CSS (Custom Properties)
- Runtime: Node.js (Express 5.x)

## Use

Build and test locally for development.
Scan, build, and deploy for test (e.g., UAT) and production.

### Development
```docker run -it --rm -p 8018:8018 web-landing:dev```

### Testing

#### 0. One-time setup
`npm ci`

#### 6. Verify the site on http://localhost:8018
